== Description ==

Pages From Category will allow you to add page links to your Sidebar.
What you can customize:
1. Which Category of pages you want to show
2. Number of page links to display
3. Display options: Date Posted, Date Modified, etc.
4. List newest to oldest or oldest to newest
5. Exclude specific pages from the list
6. Show RSS feeds

== Configuration ==

You can add Pages From Category to your site in a few easy steps.
1. Install the plugin
2. Activate the plugin
3. Go to Widget's page and add Pages From Category to sidebar
4. Edit the Settings or keep the defaults.

== Installation ==

Unzip the plugin and copy the 3mergingdesign folder to your wp-content/plugins folder or install from wordpress.org repository and activate the plugin.
